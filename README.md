### Supported features

Our diverse payment features include:

* Online payments
* Card present payments
* Storing a card on file
* Surcharge payments
* Recurring payments
* Authorizing payments
* Voiding (reversals)
* Crediting (refunds)
* Settlements
* Reporting
* Manual keyed transactions
